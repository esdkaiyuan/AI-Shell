## 🔴 严重问题

### 1. Shell数据流未连接到渲染进程
**位置**: `packages/desktop/src/main/shell/manager.ts`
**问题**: ShellManager创建了EventEmitter但没有通过IPC将数据发送到渲染进程
**影响**: 终端无法显示Shell输出
**优先级**: P0 - 阻塞性问题

**当前代码**:
```typescript
ptyProcess.onData(data => {
  emitter.emit('data', data);  // 只在主进程emit，渲染进程收不到
});
```

**需要修复**: 添加IPC通信将数据发送到渲染进程

---

### 2. SSH数据流未连接到渲染进程
**位置**: `packages/desktop/src/main/ssh/manager.ts`
**问题**: 同样的问题，SSH输出无法传递到渲染进程
**影响**: SSH终端无法显示输出
**优先级**: P0 - 阻塞性问题

---

### 3. 缺少React依赖
**位置**: `packages/desktop/package.json`
