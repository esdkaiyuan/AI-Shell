### 7. 快速开始示例

#### 配置OpenAI（推荐新手）

1. 访问 https://platform.openai.com/api-keys
2. 创建新的API密钥
3. 在AI Shell中：
   - 选择"OpenAI"卡片
   - 粘贴API Key
   - 选择"gpt-3.5-turbo"
   - 点击确定

#### 配置Ollama（免费本地）

1. 安装Ollama：
```bash
curl -fsSL https://ollama.ai/install.sh | sh
```

2. 拉取模型：
```bash
ollama pull llama2
```

3. 在AI Shell中：
   - 选择"Ollama (本地)"卡片
   - API Key留空
   - 选择"llama2"
   - 点击确定

#### 配置百度文心（国内推荐）

1. 访问百度智能云控制台
2. 获取API Key和Secret Key
3. 在AI Shell中：
   - 选择"百度文心一言"卡片
   - 输入：`API_KEY:SECRET_KEY`
   - 选择"ernie-bot-turbo"
   - 点击确定

### 8. 常见问题

**Q: 可以同时配置多个提供商吗？**
A: 可以！系统会使用第一个启用的提供商。

**Q: API Key存储在哪里？**
A: 加密存储在本地SQLite数据库中，完全安全。

**Q: 如何切换使用的AI？**
A: 删除当前提供商，添加新的即可。未来会支持快速切换。

**Q: 本地模型需要API Key吗？**
A: Ollama等本地模型不需要API Key，直接选择即可。

**Q: 配置错误怎么办？**
A: 删除后重新添加，或检查API Key格式是否正确。
