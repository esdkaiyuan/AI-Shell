## 🎯 核心架构

### 1. 终端事件总线
```typescript
terminalBus.register({
  id, insert, run, getContext
})
```
- 连接终端和 AI 面板
- 解耦组件通信
- 自动跟踪活动终端

### 2. 上下文捕获
- 20KB 滚动缓冲
- 自动去除 ANSI 码
- 实时更新

### 3. 命令提取
- 正则匹配代码块
- 启发式识别命令
- 自动生成操作按钮

## 🚀 如何使用

### 启动开发环境
```bash
cd packages/desktop
npm install
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 主要依赖
- xterm.js - 终端模拟
- xterm-addon-fit - 自适应
- xterm-addon-search - 搜索
- xterm-addon-web-links - 链接识别

## 💡 使用示例

### AI 助手
```
1. 点击左侧 AI 图标
2. 输入："这个错误怎么解决？"
3. AI 自动读取终端上下文
4. 点击"执行"按钮运行建议的命令
```

### SSH 连接
```
1. 点击工具栏 🌐 图标
2. 新建连接 → 填写服务器信息
3. 点击"连接"
4. 新标签自动打开 SSH 终端
```

### 会话管理
```
1. 打开多个终端（本地 + SSH）
2. 点击 💾 保存会话
3. 明天点击 📁 加载会话
4. 整个环境瞬间恢复
```

### 分屏
```
1. 点击 ⊞ 图标
2. 左侧运行服务器
3. 右侧执行测试
4. 同时查看两个输出
```
